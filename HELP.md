Реализовать приложение с использованием Java + Spring Boot + Angular
Backend (Java + Spring Boot + JDBC Template + СУБД): хранит в любой СУБД данные о товарах (уникальный код товара (он же уникальный id) 
- произвольный набор, допускающий использование до 32-х любых символов; 
наименование; цена; артикул; дата производства; количество), реализует REST со всеми CRUD операциями. Написать unit тесты
Frontend (Angular 11 ): отображение информации по товарам в табличном виде, все CRUD операции по уникальному коду товара (он же уникальный id). 
  Желательно не использовать сторонние библиотеки компонентов.
